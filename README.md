# MABOT - Cafe Management Backend

A professional Django-based backend system for managing cafe operations.

## Features
- PostgreSQL Database integration.
- Secure environment configuration using `python-dotenv`.
- Media file management for menu items.
- Timezone configured for Africa/Addis_Ababa.

## Setup Instructions
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`.
3. Install dependencies: `pip install -r requirements.txt`.
4. Setup your `.env` file based on the environment.
5. Run migrations: `python manage.py migrate`.
